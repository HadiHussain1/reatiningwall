Gallery Manager

Manage gallery sections and images. All changes are saved to your browser's local storage and appear immediately on the Our Work page.

How it works: Create sections (e.g. "Concrete Sleeper Projects 2024"), then add images to each section. Images are referenced by URL or file path — make sure images are placed in the same folder as this site, then use a relative path like images/my-photo.jpg. Sections appear on the gallery page in the order you create them.
